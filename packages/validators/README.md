# @workspace/validators

Zod validation schemas shared across the API and web app. Defines request/response shapes, entity schemas, and business rules.

## Exports

| Module | Description |
|--------|-------------|
| `./common` | Shared types (pagination, sorting, ID formats) |
| `./organizations` | Organization create/update schemas |
| `./products` | Product and variant schemas |
| `./inventory` | Inventory level schemas |
| `./integrations` | Shopify integration schemas |
| `./api-keys` | API key create/revoke schemas |
| `./events` | Event log schemas |
| `./feeds` | Feed and feed source schemas |
| `./sync-settings` | Sync configuration schemas |
| `./connections` | Supplier-retailer connection schemas |
| `./api` | Public API request/response schemas |
| `./application-fields` | Feed application form field definitions |
