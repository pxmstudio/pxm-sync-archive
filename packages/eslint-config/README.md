# @workspace/eslint-config

Shared ESLint configuration for the monorepo. ESLint v9 with flat config.

## Configs

| Export | Use for |
|--------|---------|
| `./base` | All packages |
| `./next-js` | Next.js apps |
| `./react-internal` | React library packages |

Includes TypeScript, React, React Hooks, Prettier, and Turbo plugins.
